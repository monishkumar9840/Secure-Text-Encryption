# 🔐 Multi-Algorithm Encryption Tool

A Python-based encryption application that demonstrates three widely used encryption algorithms:

- AES (Advanced Encryption Standard)
- DES (Data Encryption Standard)
- RSA (Rivest-Shamir-Adleman)

This project allows users to encrypt and decrypt text using symmetric and asymmetric encryption techniques.

---

## Features

✔ AES Encryption & Decryption

✔ DES Encryption & Decryption

✔ RSA Encryption & Decryption

✔ Random Key Generation

## Encryption Algorithms

### AES

- Symmetric Encryption
- 128-bit Key
- CBC Mode
- Secure and Fast

### DES

- Symmetric Encryption
- 56-bit Effective Key
- CBC Mode

### RSA

- Asymmetric Encryption
- 2048-bit Key Pair
- OAEP Padding
- SHA-256 Hash
